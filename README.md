## Project Description
A JavaScript-driven application that performs mathematical operations and manages student academic data. This project demonstrates core programming concepts including functions, conditional logic, and advanced array methods like `.reduce()` and `.filter()`.

## Technologies Used
- HTML5
- JavaScript (ES6+)
- Git Bash (Version Control)

## Features
- **Dynamic Calculator:** Supports addition, subtraction, multiplication, division, and exponentiation.
- **Error Handling:** Includes logic to prevent and notify users of division by zero.
- **Student Analytics:** Automatically calculates class averages and identifies students with scores below a specific threshold.

## How to Run
1. Clone this repository to your local machine:
https://github.com/maune-tut/iyf-s10-week-04-maune-tut
2. Open `index.html` in your browser (Right-click > Open with Google Chrome).
3. Open the Developer Console (F12) to see the output of the functions.

## Lessons Learned
- Mastered the use of the `.reduce()` method to calculate sums across an array of objects.
- Learned how to implement a `switch` statement for cleaner, more readable calculator logic compared to multiple `if/else` blocks.
- Practiced semantic documentation of code to make logic easier for others to follow.

## Challenges Faced
- **Live Server Connectivity:** Experienced a "Site Not Found" error with the VS Code Live Server extension. I resolved this by utilizing the "Reveal in File Explorer" method and manual browser refreshing to continue development without delays.
- **Array Object Manipulation:** Initially struggled with accessing nested object properties within `.map()`, but solved it by reviewing documentation on JavaScript object dot notation.
